NEW FRAMEWORK FOR IFIC STATISTICS

In an effort to preserve the value of the comprehensive data offered by IFIC Statistics, a new balance has been struck that provides greater differentiation between the data available to Contributors and Subscribers and that which can be freely accessed. Please note that fund managers will maintain their access via a new Data Contributor Agreement (as detailed in previous communications), as will Subscribers (non-contributors of data) through a Data Subscriber Agreement. Fund Managers are prohibited from being Subscribers.
On August 15, 2011, changes were made to IFIC Statistics. Industry figures now include supplementary data provided by Investor Economics, resulting in data that better reflects the current size and activity of the Canadian retail mutual fund industry.

HOW TO ACCESS IFIC STATISTICS

This new website has been built to provide Contributors and Subscribers with the ability to access the level of detail that they are seeking. Under this new Framework, each firm must designate an individual as a Person Ultimately Responsible for Statistics Entitlements who will undertake to fulfill the duties outlined in the Data Subscriber Agreement. User information reported under the Data Subscriber Agreement will be uploaded and gain access via user logins to the password protected site.

To continue access the reports mentioned above or other non-public IFIC Statistics' reports, a duly completed Data Subscriber Agreement must be received and accepted by IFIC. The necessary documents are available via the hyperlinks below.

  • Data Subscriber Agreement
  • Request Form A: DATABASE ACCESS GROUP FOR STATISTICS
  • Subscriptions Fees & Packages

Please contact IFIC's Statistics group via email at statistics@ific.ca for more details, or if you have any questions.

Market Share:

Market Share uses the firm's proprietary data compared to some industry aggregate*, e.g. CIFSC. This calculation / comparison can be performed using data released both monthly and quarterly. No knowledge of competitors' data is required.

* IFIC recommends that for external communications, participants use the Investor Economics aggregates for the computation of fund company market share statistics.

Rankings:

Any ranking requires knowledge of own firm's position relative to competition's position. Since this knowledge can only exist using Detailed Data, rankings are possible only using released Quarterly Subscriber Data.
